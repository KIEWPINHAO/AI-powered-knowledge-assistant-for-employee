# AI-powered-knowledge-assistant-for-employee
An AI-powered knowledge assistant that allows users to query internal company documents using natural language.

The system uses Retrieval-Augmented Generation (RAG) to retrieve relevant document context from a vector database and generate accurate answers using a Large Language Model.

Built with modern LLM engineering tools such as **LangChain**, **PineCone**, and **FastAPI**.

Features

## AI-powered question answering over internal documents

1. Semantic search using embeddings
2. Retrieval-Augmented Generation (RAG)
3. Source citation for responses
4. Document ingestion pipeline
5. REST API for integration
6. Scalable vector search
